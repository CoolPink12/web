CSS

CSS(Cascading Style Sheets)는 HTML 또는 XML과 같은 마크업 언어(SVG, MathML 또는 XHTML과 같은 XML 언어 포함)로 작성된 문서의 프레젠테이션 및 스타일을 지정하는 데 사용되는 스타일시트 언어입니다. [1] CSS는 HTML 및 JavaScript와 함께 월드 와이드 웹의 초석 기술입니다. [2] CSS는 레이아웃, 색상 및 글꼴을 포함하여 콘텐츠와 프레젠테이션을 분리할 수 있도록 설계되었습니다. [3] 이러한 분리는 콘텐츠 접근성을 향상시킬 수 있습니다. [추가 설명 필요] 프리젠테이션 특성의 사양에 더 많은 유연성과 제어를 제공합니다. 별도의 .css 파일에 관련 CSS를 지정하여 여러 웹 페이지에서 서식을 공유할 수 있도록 하여 구조적 콘텐츠의 복잡성과 반복을 줄일 수 있습니다. .css 파일을 캐시할 수 있도록 하여 파일을 공유하는 페이지와 해당 서식 간의 페이지 로드 속도를 향상시킵니다. 또한 서식과 콘텐츠를 분리하면 화면, 인쇄물, 음성(음성 기반 브라우저 또는 화면 읽기 프로그램을 통해) 및 점자 기반 촉각 장치와 같은 다양한 렌더링 방법에 대해 동일한 태그 페이지를 다양한 스타일로 표시할 수 있습니다. CSS에는 모바일 장치에서 콘텐츠에 액세스하는 경우 대체 서식에 대한 규칙도 있습니다. [4] 연계 이름은 둘 이상의 속성 선언이 특정 요소와 일치하는 경우 적용되는 선언을 결정하기 위해 지정된 우선 순위 체계에서 비롯됩니다. 이 연계 우선 순위 체계는 예측할 수 있습니다. CSS 사양은 W3C(World Wide Web 컨소시엄)에서 유지 관리합니다. 인터넷 미디어 유형(MIME 유형)은 RFC 2318(1998년 3월)에 의해 CSS와 함께 사용하도록 등록되었습니다. W3C는 CSS 문서에 대한 무료 CSS 유효성 검사 서비스를 운영합니다. [5]text/css HTML 외에도 XHTML, 일반 XML, SVG 및 XUL을 포함한 다른 마크업 언어가 CSS 사용을 지원합니다. CSS는 GTK 위젯 툴킷에서도 사용됩니다.
