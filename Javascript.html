JavaScript

종종 JS로 약칭되는 JavaScript (/ ˈdʒɑːvəskrɪpt /)는 HTML 및 CSS와 함께 월드 와이드 웹의 프로그래밍 언어이자 핵심 기술입니다. 2024년 현재, 웹 사이트의 98.9 %는 웹 페이지 동작을 위해 클라이언트 측에서 JavaScript를 사용하며,[10] 종종 타사 라이브러리를 통합합니다. 모든 주요 웹 브라우저에는 사용자 장치에서 코드를 실행하는 전용 JavaScript 엔진이 있습니다. JavaScript는 ECMAScript 표준을 준수하는 높은 수준의 JIT(Just-In-Time) 컴파일 언어입니다. [11] 동적 타이핑, 프로토타입 기반 객체 지향 및 일류 기능이 있습니다. 다중 패러다임이며 이벤트 기반, 기능적 및 명령적 프로그래밍 스타일을 지원합니다. 텍스트, 날짜, 정규식, 표준 데이터 구조 및 DOM(문서 개체 모델)으로 작업하기 위한 API(응용 프로그래밍 인터페이스)가 있습니다. ECMAScript 표준에는 네트워킹, 스토리지 또는 그래픽 기능과 같은 I/O(입력/출력)가 포함되어 있지 않습니다. 실제로 웹 브라우저 또는 기타 런타임 시스템은 I/O용 JavaScript API를 제공합니다. JavaScript 엔진은 원래 웹 브라우저에서만 사용되었지만 이제는 일부 서버 및 다양한 응용 프로그램의 핵심 구성 요소입니다. 이 용도에 가장 많이 사용되는 런타임 시스템은 Node.js입니다. Java와 JavaScript는 이름, 구문 및 각각의 표준 라이브러리가 유사하지만 두 언어는 별개이며 디자인이 크게 다릅니다
